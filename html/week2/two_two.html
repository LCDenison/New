Colloquialism and Slang

Function

  • ! The dictionary provides current colloquial expressions.
  • The translation site provides clear indications about how often (relative frequency) colloquial expressions are used.
  • The translation site can process specific expression searche queries.

Sources

  • The translation site gathers material from writing.
  • The translation site gathers material from audio/movie/tv transcripts.
  • The translation site gathers material from social media sites.
  • The site provides the opportunity to find more detail on all terms.
  • The site indicates a link to the sources referenced.
  • The site indicates the source (social media/movie) with icons or another easy to read method.

Processing

  • The dictionary provides aggregated data about the location of the colloquialisms.
  • The dictionary provides lexical variants, broken down by location.

Extra features

  • The translation site links to a defnition dictionary.
  • The translation site links to individual words in more extensive context.
  • The translation site visually links similar/related terms.
  • The translation site links to/possesses the ability to pronounce expressions/words phoenetically.

Web design

  • The translation site is easy to search.
  • The search box is clearly indicated.
  • The translation site can be used on the Internet.
  • The translation site can be used on a mobile.
  • The translation site provides a visual indicator of how often colloquial expressions are used.
  • The site highlights the relevant term in contextual examples.

Organization

  • The translation site offers way to search expressions without looking up the specificexpression (like a print phrase book).
  • ! The translation site groups expressions with similar meanings, themes, or contexts together.
  • The user can browse by these themes or contexts in addition to browsing by typing the word/expression in.

UX

  • The translation site has auto completion.
  • The translation site is hyperlinked.
  • The site offers all given information about the word/expression in the result window of the search box.
  • The site returns results quickly.
  • The site returns a result with several examples in context.
